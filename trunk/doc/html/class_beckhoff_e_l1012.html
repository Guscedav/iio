Detailed Description

This class implements a device driver for the Beckhoff EL1012 digital input module. The EL1012 is a module that offers 2 digital inputs at a level of 24 V.


The Beckhoff EL1012 digital input module


An example that shows how to use this device driver is given below: 


EtherCAT etherCAT("192.168.1.130");  // create EtherCAT object with given interface
CoE coe(etherCAT, 0.005);            // create CANopen over EtherCAT driver
BeckhoffEL1012 beckhoffEL1012(etherCAT, coe, 0x0000); // create EL1012 driver
DigitalIn digitalIn0(beckhoffEL1012, 0);  // create the digital inputs
DigitalIn digitalIn1(beckhoffEL1012, 1);
if (digitalIn0) {  // check the input 0
    ...
}
bool value = digitalIn2;  // read the input 1
...

Also see the documentation about the EtherCAT and the CoE classes for more information.
