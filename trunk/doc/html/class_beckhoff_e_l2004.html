Detailed Description

This class implements a device driver for the Beckhoff EL2004 digital output module. The EL2004 is a module that offers 4 digital outputs at a level of 24 V.


The Beckhoff EL2004 digital output module


An example that shows how to use this device driver is given below: 


EtherCAT etherCAT("192.168.1.130");  // create EtherCAT object with given interface
CoE coe(etherCAT, 0.005);            // create CANopen over EtherCAT driver
BeckhoffEL2004 beckhoffEL2004(etherCAT, coe, 0x0000); // create EL2004 driver
DigitalOut digitalOut0(beckhoffEL2004, 0);  // create the digital outputs
DigitalOut digitalOut1(beckhoffEL2004, 1);
DigitalOut digitalOut2(beckhoffEL2004, 2);
DigitalOut digitalOut3(beckhoffEL2004, 3);
digitalOut0 = true;       // set the output to logical 1
digitalOut1 = false;      // set the output to logical 0
...

Also see the documentation about the EtherCAT and the CoE classes for more information.
