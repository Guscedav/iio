Detailed Description

This class implements a device driver for the Beckhoff EL2258 digital output module. The EL2258 is a module that offers 8 digital outputs at a level of 24 V.


The Beckhoff EL2258 digital output module


An example that shows how to use this device driver is given below: 


EtherCAT etherCAT("192.168.1.130");  // create EtherCAT object with given interface
CoE coe(etherCAT, 0.005);            // create CANopen over EtherCAT driver
BeckhoffEL2258 beckhoffEL2258(etherCAT, coe, 0xFFFF, 1000); // create EL2258 driver
DigitalOut digitalOut0(beckhoffEL2258, 0);  // create the digital outputs
DigitalOut digitalOut1(beckhoffEL2258, 1);
DigitalOut digitalOut2(beckhoffEL2258, 2);
DigitalOut digitalOut3(beckhoffEL2258, 3);
DigitalOut digitalOut4(beckhoffEL2258, 4);
DigitalOut digitalOut5(beckhoffEL2258, 5);
DigitalOut digitalOut6(beckhoffEL2258, 6);
DigitalOut digitalOut7(beckhoffEL2258, 7);
digitalOut0 = true;       // set the output to logical 1
digitalOut1 = false;      // set the output to logical 0
...

Also see the documentation about the EtherCAT and the CoE classes for more information.
