Detailed Description

The CANopenMailboxDatagram is an extension of the MailboxDatagram for CANopen over EtherCAT mailbox communication.

Creates an EtherCAT datagram for CANopen over EtherCAT mailbox communication.

Parameters
commandthe command of this datagram, i.e. COMMAND_APRD or COMMAND_APWR.
deviceAddressthe relative device address, i.e. 0x0000, 0xFFFF, 0xFFFE, etc.
offsetAddressthe address within the EtherCAT slave controller, usually the address in the process data RAM of the EtherCAT slave controller for mailbox communication.
mailboxLengththe length of the entire payload of this datagram.
messageTypethe CANopen message type, i.e. MESSAGE_TYPE_EMERGENCY_MESSAGE, MESSAGE_TYPE_SDO_REQUEST or MESSAGE_TYPE_SDO_RESPONSE.
indexthe index of the CANopen service data object (16 bit).
subindexthe subindex of the CANopen service data object (8 bit).
valuethe value to write into this CANopen mailbox datagram (8 - 32 bit).
lengththe number of bytes the value consists of, usually 1, 2 or 4.
