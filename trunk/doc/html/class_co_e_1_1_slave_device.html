Detailed Description

The SlaveDevice class offers callback methods for an EtherCAT master. It needs to be implemented by EtherCAT slave device drivers.

void CoE::SlaveDevice::readDatagram ( )
virtual

This method is called by the communication handler just after a new EtherCAT frame was received from the fieldbus. It allows a slave device driver to read its datagrams with input process data.

void CoE::SlaveDevice::writeDatagram ( )
virtual

This method is called by the communication handler just before a new EtherCAT frame is transmitted on the fieldbus. It allows a slave device driver to update its datagrams with output process data.
