Detailed Description

This class implements a simple device driver for the Slamtec RoboPeak lidar version A. It receives measurements from the serial interface and sends these measurements to a registered delegate object.


The Slamtec RoboPeak lidar Version A


Another object that wishes to receive measurements from this device driver must implement the receiveMeasurement() method of the delegate class. An example of a user class that implements this method is given below:
