Detailed Description

This class implements a specific CAN device driver for the Peak System PCAN-PCIexpress mini interface board. The PCAN-PCIexpress mini board features two SJA1000 CAN bus controllers that allow to transmit and receive CAN 2.0B compliant frames. They offer a receive buffer with a size of 64 bytes.

The Peak PCAN-PCIexpress mini CAN interface board
